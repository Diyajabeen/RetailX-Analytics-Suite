📊 RetailX Analytics Suite

A Business Intelligence project built using Microsoft Excel and Power BI to analyze sales performance, customer behavior, and inventory operations through interactive dashboards and KPI tracking.

🚀 Features

- 📈 Executive Overview Dashboard
- 💰 Sales Performance Analysis
- 👥 Customer Insights Dashboard
- 📦 Inventory & Operations Monitoring
- 🎯 Revenue Target vs Achievement Analysis
- 📊 KPI Tracking with DAX Measures
- 🏗️ Star Schema Data Modeling

🛠️ Tech Stack

- 📑 Microsoft Excel
- 📊 Power BI
- ⚡ DAX (Data Analysis Expressions)
- 🏗️ Data Modeling

📋 Dashboard Pages

1. 📈 Executive Overview
2. 💰 Sales Performance
3. 👥 Customer Insights
4. 📦 Inventory & Operations

🎯 Key Metrics

- 💵 Total Revenue
- 📦 Total Orders
- 👥 Total Customers
- 💹 Total Profit
- 📊 Profit Margin %
- 🎯 Revenue Target Achievement
- 📦 Inventory Utilization

📌 Project Outcome

Developed a centralized business intelligence solution that transforms raw retail data into actionable insights, enabling stakeholders to monitor KPIs, analyze trends, and make data-driven decisions efficiently.
